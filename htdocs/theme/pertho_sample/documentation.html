  • 01. About Pertho Admin template.Open or Close

    Pertho Admin is a HTML5/CSS (CSS3) admin backend template build on Foundation framework enchanced with many third party components.

    Here you can find extended documentation for this framework.

    JS/jQuery plugin used in this template:

    • Accordion Nested Accordion
    • Autocomplete jQuery Autocomplete Plugin
    • Autosize jQuery plugin to enable automatic height for textarea elements
    • Calendar jQuery Plugin FullCalendar
    • Chosen Select box enhancement plugin
    • Color Picker jQuery colorpicker
    • Data Tables Dynamic tables
    • Elfinder File explorer
    • Fancybox tool for displaying images, HTML content and multi-media
    • File uploader A highly usable upload handler for your Content Management Systems or similar
    • Flexible Accordion Flexible Slide-to-top Accordion
    • Get hidden width/height jQuery plugin that gets the actual width of hidden DOM elements
    • hoverIntent plug-in that attempts to determine the user's intent
    • iButtons iPhone-style buttons
    • Input Mask jQuery Input Mask
    • jQuery JS framework
    • jQuery Easing jQuery easing
    • jQuery Tools jQuery Tools
    • jQuery UI user interface library
    • jQplot charts JS charting component
    • jQuery metadata Extracting metadata from classes, random attributes, and child elements
    • JS validation jQuery plugin: Validation
    • Mobile Navigation jQuery Mobile Select
    • Placeholder plugin that enables HTML5 placeholder behavior
    • Mobile table Rwd table.
    • Smart resize Debounced Resize Event for jQuery
    • Rating A Star Rating Plugin
    • serializeObject Serialize a form into an object
    • showMessage showMessage jquery plugin
    • Smooth Navigational Menu Smooth Navigation Menu is a multi level, CSS list based menu
    • Sticky A super simple notification system for jQuery
    • Super flexible HTML lists searchable, sortable and filterable list
    • Tag Handler managing tag-type metadata
    • Timepicker timepicker addon to jQuery UI datepicker
    • Tiny MCE Javascript HTML WYSIWYG editor
    • Textarea Limiter jQuery plugin that provides a character counter
    • Tooltips qtip2 tooltips
    • UItoTop UItoTop jQuery Plugin
    • Wizard Customizable wizard

  • 06. Colors, backgrounds, gradientsOpen or Close
    Backgrounds:
    Add one of the following class to <body> or any block element to change background.
    • .ptrn_a
    • .ptrn_b
    • .ptrn_c
    • .ptrn_d
    • .ptrn_e
    Bar gradients:
    Add one of the following class to <body> to change bar gradient.
    • .grdnt_a
    • .grdnt_b
    • .grdnt_c
    • .grdnt_d
    • .grdnt_e
    Menu hover colors:
    Add one of the following class to <body> to change main navigation hover color.
    • .mhover_a
    • .mhover_b
    • .mhover_c
    • .mhover_d
    • .mhover_e
    Custom colors:
    Add your custom colors to <header> section: or add them at the end of main stylesheet - css/style.css
  • 07. jQuery plugins.Open or Close

    You can find extended documentation for plugins on plugin pages.

    Plugins are split into 2 folders:

    • js/ folder contains small scripts (jQuery tools, listnav, sparklines, etc.), combined scripts (s_scripts.js), jQuery framework, jQuery UI, and configuration file (pertho.js).
    • lib/ folder contains more complex plugins (TinyMCE, Elfinder, Datatables etc.).

    Configuration for all the plugins is in js/pertho.js.
